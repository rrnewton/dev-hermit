# Experiments

Store durable, reproducible experiment evidence in a descriptive subdirectory.
Each experiment should document its question, method, commands, exact source
commits, environment, and result. Keep loose exploratory output in `scratch/`.

Available experiments:

- `qemu_under_hermit_20260721/`: QEMU TCG boot attempts under Hermit, including
  exact environment metadata, the result matrix, and failure analysis.
